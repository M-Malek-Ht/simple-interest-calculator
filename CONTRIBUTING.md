# Contributing

Thank you for your interest in contributing to this project.

All contributions are welcome. This includes bug reports, bug fixes, documentation improvements, enhancements, and new ideas.

## How to Contribute

You can contribute by:

- Reporting bugs
- Fixing bugs
- Improving documentation
- Suggesting enhancements
- Sharing new ideas
- Improving the simple interest calculator script

## Bug Reports

If you find a bug, please open an issue and describe the problem clearly. Include steps to reproduce the issue if possible.

## Bug Fixes

Bug fixes are welcome. Please explain what problem your fix solves when submitting your changes.

## Documentation Improvements

Documentation improvements are encouraged. You can help by improving explanations, fixing spelling mistakes, or making instructions clearer.

## Enhancements and Ideas

New enhancements and ideas are welcome. If you have a suggestion to improve the project, please open an issue or submit a pull request.

## Pull Requests

Before submitting a pull request, make sure your changes are clear and tested. Please provide a short description of what you changed.

## Thank You

Thank you for helping improve this project.
